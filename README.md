# SQL-Pizza-Sales-Project
SQL-based Pizza Sales Analysis project using MySQL to analyze revenue, sales trends, customer ordering patterns, and business insights through complex queries.

## Skills Demonstrated
- SQL Joins
- Aggregate Functions
- GROUP BY & HAVING
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- Date & Time Functions

## Key Business Questions Solved
- Total revenue generated
- Average order value
- Total pizzas sold
- Daily & hourly order trends
- Best and worst selling pizzas
- Revenue by pizza category and size
- Percentage contribution of each category
- Cumulative revenue over time
- Top 3 pizzas by revenue within each category

## Database
- orders
- order_details
- pizzas
- pizza_types

## Tools Used
- MySQL
- MySQL Workbench
